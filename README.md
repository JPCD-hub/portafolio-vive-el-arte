# Portafolio Vive El Arte

Pagina web estatica para el portafolio de eventos de Vive El Arte.

Incluye:

- Catalogo de eventos realizados.
- Ficha individual por evento.
- Flyers y registro fotografico.
- Formulario de contacto con salida a WhatsApp.

## Ver Localmente

Abre `index.html` en el navegador o ejecuta un servidor local:

```powershell
python -m http.server 5500
```

Luego entra a:

```text
http://127.0.0.1:5500/index.html
```
